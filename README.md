# Student Marks Analysis using Python (Pandas)

This project analyzes student marks using Python and the pandas library.

## Features

- Reads student dataset from CSV
- Displays full dataset and first few students
- Calculates average marks for each subject
- Finds top scorer in Maths
- Filters students scoring below 70 in Maths
- Filters students scoring above 85 in Science
- Calculates total marks and finds the highest total scorer

## How to Run

1. Download `students.csv` and `student_analysis.py` from this repository
2. Make sure Python and pandas are installed
3. Run the script:

```bash
python student_analysis.py
