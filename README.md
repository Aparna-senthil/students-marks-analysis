# Student Marks Analysis using Python (Pandas)

This project analyzes student marks using Python and pandas:

- Reads student dataset from CSV
- Displays full dataset and first 3 students
- Calculates average marks
- Finds top scorer in Maths
- Filters students scoring below 70 in Maths
- Filters students scoring above 85 in Science
- Calculates total marks and finds highest total scorer
